# prettier-config

Common Prettier configuration used in Hamster

## Install

```bash
npm i --save-dev @hamster-bot/prettier-config
yarn add --dev npm i @hamster-bot/prettier-config
```

## Use

```jsonc
// package.json
{
  "prettier": "@hamster-bot/prettier-config"
}
```
